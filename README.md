## English Janala
 It's an interactive vocabulary learning platform that dynamically loads data from APIs and presents it in a user-friendly interface.
---

##Live Link
https://ashrafulhoda789.github.io/English-Janala-Project-PH/

---

## Project Features:
• Dynamic lesson levels generated from API
• Vocabulary cards loaded based on selected lesson
• Active lesson highlighting for better navigation
• Vocabulary details modal with pronunciation, example, and synonyms
• Real-time search functionality
• Save vocabulary feature for quick review
• Voice pronunciation using the Speech Synthesis API
• Loading spinner and proper empty/error state handling

## Tech Stack: 
• HTML5
• Tailwind CSS
• DaisyUI
• JavaScript
• Document Object Manipulatio (DOM)
• REST APIs
